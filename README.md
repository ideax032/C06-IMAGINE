# C06-IMAGINE – Learning Tasks Repository

This repository contains the structured learning tasks for the **IMAGINE: Sketch-to-Image Generation with Autoencoder-based Enhancement** project.

The purpose of this repository is to build strong theoretical foundations before moving into neural networks, autoencoders, and the multimodal pipeline.

This is not a coding-first repository. It is a concept-first repository.

---

## How This Repository Works

Each task exists inside its own folder.

Inside every task folder, you will find:

- `README.md` containing:
  - Learning objectives  
  - Concepts to understand  
  - Required reading topics  
  - Expected conceptual outcomes  

First fork the repository and after completing a task, you must create:

- `explanation.md` inside that same task folder.

In `explanation.md`, write briefly:

- What you understood from the concepts
- Important ideas you learned
- Any confusion or difficulty faced (optional)

---

## Guidelines

- Prioritize understanding over implementation.
- Use clean reasoning and clear explanations.
- Keep explanations concise and written in your own words.
- Push your progress regularly.

---

## Learning Phases

This repository follows the IMAGINE learning roadmap:

- Learning Phase 1 – Machine Learning Foundations  
- Learning Phase 2 – Neural Networks  
- Learning Phase 3 – Autoencoders and Diffusion Models  
- Practical Build Phases – Integration and Deployment  

Each phase prepares you for the next. Do not skip phases.

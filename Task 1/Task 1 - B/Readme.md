# Task 1B – Learning Phase 1 B (Once Comfortable with Python)

## Objective

This task is for students who already know Python basics but need structured exposure to Machine Learning foundations.

The focus of this phase is:

- Understanding core machine learning concepts
- Understanding how models learn
- Basic data handling using NumPy and Pandas
- Small demonstrations (not full model implementations)

This task is conceptual-first with minimal implementation.

---

## Learning Outcomes

By the end of this task, you should be able to:

- Explain what supervised learning is
- Differentiate between regression and classification
- Understand what a loss function represents
- Explain gradient descent intuitively
- Understand how data is structured for ML
- Use NumPy and Pandas for dataset handling


---

## Concepts to Study

### 1. What is Machine Learning?

Understand:

- Difference between traditional programming and ML
- What is a dataset?
- Features vs Labels
- Training data vs Testing data
- Model as a function approximation

You must be able to explain ML in simple technical terms.

---

### 2. Supervised Learning

Study:

- Regression problems
- Classification problems
- Examples of real-world use cases
- How predictions are evaluated

Understand conceptually:

- What is a hypothesis?
- What is prediction error?
- Why do we need optimization?

---

### 3. Loss Functions

Understand the purpose of:

- Mean Squared Error (for regression)
- Binary Cross Entropy (for classification)

Conceptual Questions you must answer in explanation.md:

- What does a loss function measure?
- Why does minimizing loss improve performance?
- Why are different losses used for different tasks?

---

### 4. Gradient Descent (Conceptual Only)

You are not required to fully implement gradient descent.

You must understand:

- What is an optimization problem?
- Why do we compute gradients?
- What does "moving in the negative gradient direction" mean?
- Learning rate and its effect

Be able to explain the intuition without heavy mathematics.

---

## Basic Practical Component

This phase requires only basic exposure to NumPy and Pandas.

### NumPy Tasks

- Create arrays
- Perform element-wise operations
- Perform matrix multiplication
- Understand array shapes
- Demonstrate simple vectorized computation

No ML models need to be built.

---
### Pandas Tasks

- Load a CSV dataset
- Inspect dataset structure
- Handle missing values (basic)
- Select specific columns
- Perform simple descriptive statistics

Focus on understanding how data is prepared before modeling.


---

## Deliverables

Inside the `Task1B/` folder, you must include:

- A notebook or `.py` file demonstrating NumPy and Pandas basics
- `explanation.md`

Your `explanation.md` must clearly explain:

- Your understanding of supervised learning
- The difference between regression and classification
- What a loss function represents
- What gradient descent does conceptually
- How NumPy and Pandas help in ML workflows

---

## Expected Outcome

After completing this task, you should:

- Understand how learning happens at a high level
- Be comfortable handling structured datasets
- Be prepared to move into Task 1 - C in Learning Phase 1

This phase is about conceptual clarity, not advanced implementation.
